# Humanize ES Markers

Lee este archivo solo cuando el texto este en espanol o mezclado con espanol.

## Senales fuertes de vocabulario

Reemplazar casi siempre si suenan decorativas o comodin:

```text
abordar, ambito, panorama (metaforico), multifacetico, matizado, apalancarse,
potenciar, paradigma, holistico, sinergia, robusto fuera de contexto tecnico,
integral como relleno, transversal, vertebrar, articular como metafora,
pivotar, disruptivo, escalable fuera de tech, ecosistema fuera de biologia,
empoderar, visibilizar, alineamiento, impulsar como comodin
```

## Frases formulaicas

Eliminar o reestructurar cuando aparezcan como plantilla:

```text
Es importante senalar que...

En la actualidad...

En este sentido...
De esta manera...
Por ende...

Resulta pertinente...
Es preciso mencionar...
Es fundamental tener en cuenta que...
Con respecto a lo anterior...
En este orden de ideas...
```

## Vicios gramaticales frecuentes

- Gerundios encadenados: demasiadas acciones enlazadas con `-ando` o `-iendo`.
- Nominalizacion excesiva: verbos convertidos en sustantivos abstractos.
- Abuso de `el cual`, `la cual`, `los cuales`.
- Voz pasiva con `se` en serie.
- Conectores apilados como `asimismo`, `no obstante`, `en este sentido`.

## Patrones de estructura y tono

- Parrafos demasiado uniformes.
- Cierres tipo resumen con `en conclusion`, `en definitiva`.
- Neutralidad artificial: nunca toma postura.
- Hedging vacio: `podria`, `de alguna manera`, `eventualmente`.
- Formalidad innecesaria: `se procedio a realizar` en vez de `hicimos`.
