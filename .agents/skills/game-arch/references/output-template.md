# Game Architecture Output Template

Lee este archivo solo cuando vayas a escribir la spec o reporte final.

## Si es proyecto nuevo

Incluye:
- project structure
- game loop
- entity model
- state machine
- screen stack
- command system
- save system
- render pipeline

## Si es proyecto existente

Incluye findings con severidad:
- CRITICO
- WARNING
- INFO

## Rendering checklist

- Un batch por frame
- Render order claro
- Camera follow/resize definido
- TextureAtlas y filtering adecuados

## Cierre sugerido

- Recomendar `$plan` para convertir la arquitectura en tareas
