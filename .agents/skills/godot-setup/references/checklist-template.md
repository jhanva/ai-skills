```markdown
# Godot Project Setup Checklist

## 1. Folder Structure
- src/
- scenes/
- assets/
- tests/
- builds/

## 2. Autoloads
- EventBus

- AudioManager

## 3. Input Actions
- Movimiento
- Accion principal
- Interact
- Pause
- Navegacion UI

## 4. Display Settings
- Viewport
- Window size
- Stretch mode
- Integer scaling
- Pixel snap
- Texture filter nearest

## 5. Audio Buses
- Master
- SFX
- Music

## 6. Export Presets
- Windows Desktop
- HTML5 opcional

## 7. Testing
- Test scene
- Input
- Autoloads
- Save/load
- Audio
- Export build
```

## Validaciones finales

- Folder structure cubre todos los asset types
- Autoloads cubren funcionalidad core
- Input actions cubren movimiento + acciones principales
- Display settings son apropiados para pixel art
- Hay al menos un export preset funcional
