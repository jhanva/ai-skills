# Scope Check Report Template

Lee este archivo solo cuando vayas a producir el reporte final en conversacion o cuando necesites el formato completo.

## Reporte sugerido

```markdown
# Scope Check Report

## Summary
[Tabla de metricas]

## Veredicto
ON TRACK / AT RISK / OVER-SCOPED

## Riesgos Identificados
[Tabla riesgo / severity / mitigacion]

## Recomendaciones
[Cuts concretos, backlog moves, acciones inmediatas]

## Proximos pasos
- Ajustar game concept
- Mover backlog
- Completar GDDs faltantes
- Planear siguiente sprint
```

## Edge cases

- Sin sprints: usar velocity default y advertir
- Sin deadline: omitir deficit y basarse en riesgos
- Proyecto demasiado nuevo: advertir precision baja
- Velocity muy variable: usar referencia conservadora
