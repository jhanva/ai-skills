# Level Designer Playbook

Lee este archivo cuando necesites heuristicas de layout, pacing o encounter placement.

## Evalua o diseña contra

- critical path claro
- exploracion opcional con recompensa
- tension curve legible
- difficulty progression
- landmarks y lectura espacial

## Reglas

- evitar backtracking sin valor
- dead ends solo si pagan con reward o info
- cada encounter debe tener funcion de pacing o aprendizaje
- pedir requirements de tileset y audio cuando afecten implementacion

## Entregable sugerido

- overview del nivel
- layout ASCII o esquema
- flow
- encounters
- tension curve
- requirements de tileset/audio
- checklist de implementacion
