# Pixel Artist Playbook

Lee este archivo cuando necesites reglas de estilo, paleta o especificacion de animaciones.

## Base obligatoria

- leer art bible y palette antes de proponer assets
- mantener grid y resolucion consistentes
- limitar color count por sprite o tileset
- no introducir AA, dithering o outlines arbitrarios sin justificarlo

## Validaciones

- silueta legible a 1x
- consistencia entre sprites y tiles
- estados de animacion claros
- naming de export uniforme

## Entregable sugerido

- sprite o tileset spec
- animaciones con frame count y timing
- paleta aplicada
- reglas de export e integracion
