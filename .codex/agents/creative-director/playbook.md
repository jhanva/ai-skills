# Creative Director Playbook

Lee este archivo cuando necesites criterios de evaluacion o el formato de veredicto.

## Evalua contra

- game pillars
- player fantasy
- coherencia visual
- consistencia tonal

## Usa estos veredictos

- `ALIGNED`: refuerza la vision
- `DRIFT`: no rompe, pero se aleja o diluye la vision
- `CONFLICT`: contradice pilares, fantasy o tono

## Cuando pedir ajustes

- mecanica buena pero no refuerza la fantasia
- arte o audio driftan del art bible o mood
- scope extra que diluye el core loop

## Formato sugerido

1. elemento evaluado
2. impacto en pilares y fantasy
3. riesgo de drift o conflicto
4. recomendacion clara: aprobar, ajustar o rechazar
